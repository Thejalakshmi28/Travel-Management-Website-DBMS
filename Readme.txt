How to run the Travel Management System(travelmanagementsystem.zip)

1.Open the file

2.Extract the travelmanagementsystem.zip zip file

2.Copy travelmanagementsystem folder

3.Paste inside root directory(for xampp xampp/htdocs)

4.Open xampp and start Apache and MySQL

5.Open PHPMyAdmin (http://localhost/phpmyadmin)

6.Create a database with name travelmanagement

7.Import travelmanagement.sql file(given inside folder)

8.Run the script http://localhost/travelmanagementsystem/loginn.html
